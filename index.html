TECNOLOGÍAS WEB

Autor: Roberto Córdova

Este post contiene: Herramientas para el diseño web

Hay numerosas tecnologías para Desarrollo Web y, a veces, es complicado para las empresas del sector elegir la mejor opción para sus objetivos. Esto se debe a que el desarrollo web tiene diferentes formas, desde páginas web convencionales hasta CRM, paneles de administración y softwares más complejos. En este post, conocerás las mejores tecnologías para Desarrollo Web en sus distintas vertientes.

Los nuevos elementos HTML5

Etiqueta Descripción
article Define un artículo de un documento.
aside Define contenido aparte del contenido de Página.
bdi Aisla una parte de texto que podría formateada en una dirección diferente de otro texto fuera de el.
details Define detalles adicionales para el usuario puede ver y ocultar.
dialog Define una caja o ventana de dialogo.
figcaption Define una captura (titulo o descripción) para un elemento figure.
figure Define el contenido (o marco) de una imagen.
footer Define un pie de página para un documento o sección.
header Define un encabezado para un documento o sección.
main Define el contenido principal de un documento.
mark Define un texto marcado o resaltado.
meter Define una escala de medición dentro del rango (un indicador).
nav Define ligas o enlaces de navegación.
progress Representa el proceso de una tarea.
rp Define una traducción o pronunciación de caracteres para tipografía de Asia Oriental.
rubi Define una notación de rubí para tipografía de Asia Oriental.
section Define una sección de un documento.
summary Define un encabezado visible para un elemento details.
time Define un date/time.
wbr Define un posible salto de línea.
